Unauthorized NextGEOSS Data Hub User

While everyone may create an account via the NextGEOSS User Management portal, only participants in the closed beta may access the data hub. If you are able to log in via the NextGEOSS User Management portal but see an "unauthorized user" message when trying to access the Data Hub, contact NextGEOSS to request access to the closed beta.

Once the closed beta is completed, the data hub will be open to all users. Use of the open data hub will not require an account.

After you have received confirmation that you may access the closed beta you may try to log in again.
